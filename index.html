スマホとPCでカレンダー・復習を共有（Supabase）

ブラウザの保存は端末ごとに別です。Supabase（無料）でプロジェクトを作り、 supabase/setup.sql を SQL エディタで実行 → 同じページの Project URL と Publishable key（旧ダッシュボードでは anon public）を下に貼り、「同期をオン」にすると、どの端末から開いても同じ丸・進捗が揃います。 Secret key は使いません。スマホだけ 401 になるときは、キー欄にパスワード自動入力が入っていないか確認し、Supabase の画面からコピーし直して保存してください（PC とブラウザの保存は別です）。

同期を使わない場合: 学習記録は JSON に書き出しで別端末へ。表だけは CSV でも保存できます。
